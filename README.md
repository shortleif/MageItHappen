# MageItHappen

A specialized Arcane Mage assistant for World of Warcraft.

## Overview

MageItHappen is a comprehensive UI and utility suite designed specifically for Arcane Mages. It focuses on optimizing gameplay through:

- **Arcane-specific tracking**: Rotation trackers, Arcane Blast counters, and proc monitoring.
- **Combat Assistance**: Time-to-Die (TTD) calculations, Cooldown tracking, and Trinket synchronization.
- **Visual Enhancements**: Customizable status bars, cast bars, and unit frames tailored for Arcane gameplay.
- **Utility**: Missing buff indicators, absorb displays, and consumable tracking.

## Features

- **Dynamic TTD (Time-To-Die)**: Real-time calculation to assist in burn phase management.
- **Rotation Tracker**: Intelligent displays to help maintain optimal spell priority.
- **Cooldown Syncing**: Automated trinket glow logic based on Arcane Power and Icy Veins cooldowns.
- **Comprehensive UI**: Clean, modular components for a professional raiding HUD.

## Installation

1. Download the latest version of the addon.
2. Navigate to your World of Warcraft installation folder: `_anniversary_\Interface\AddOns\`.
3. Extract the `MageItHappen` folder into the `AddOns` directory.
4. Launch World of Warcraft. The addon should load automatically.
